Speech-To-Emotion (STE) is an application that detects emotions from speech. An LSTM model was used to predict the emotions.
Detects seven emotions: angry, sad, neutral, happy, surprised, fearful, disgust

Video demo of application: https://vimeo.com/1174991343

# Functionality:

1.) Transcribes audio speech into text

2.) Predicts the emotions expressed in each chunk of speech

3.) Tallies the overall detected emotions across the entire audio sample

# Requirements:

1.) Microphone for live audio input

2.) .wav or .mp3 for audio file input

# Application

- Executed by running the Application.ipynb with a jupyter notebook
- GUI was created using Tkinter

# Sample  Files:
- The application can process any wav/mp3 that contains speech data.
- The sample files folder contains some audio clips samples to test out the application
